# Cold Memory

Long-term learnings for this repository.

## Source discipline

- Check company responses across multiple outlets before writing “무응답.”
- Treat lawsuits as allegations unless independently confirmed elsewhere.
- Prefer primary source documents for dates, numbers, and legal language.

## Draft quality

- Slide 1 should create a need to read slide 2.
- One reveal per slide is the safest default.
- Calm hooks outperform over-explained hooks for this account.
