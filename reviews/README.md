# Reviews Archive

Each serious thread run should create a topic-specific folder here.

Recommended structure:

```text
reviews/
  YYYY-MM-DD__topic-slug/
    fact-pack.json
    verifier-a.json
    verifier-b.json
    reconcile.json
    critic.json
```

Purpose:

- auditability
- debugging missed fact-checks
- learning from repeated hook or pacing failures
- preserving why a final draft was trusted
